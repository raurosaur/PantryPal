# PantryPal

PantryPal is a recipe search and shopping list web app that helps users search for recipes, view recipe details, build shopping lists from ingredients, and share temporary shopping lists with others.

## Live Demo

[View PantryPal](https://raurosaur.github.io/PantryPal/)

## Features

- Search recipes using the Edamam Recipe API
- View recipe details, ingredients, source, image, link
- Add ingredients to a shopping list by double clicking
- Save shopping lists locally in the browser
- Share shopping lists using backend-generated links backed by AWS SAM
- Shared lists expire automatically after 3 days
- Responsive UI built with Tailwind CSS
